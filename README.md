# Smashify - Badminton Score Tracker 🏸

A modern, dark-themed badminton scoring app for singles and doubles matches.

## Features
- 🎯 Live scoring (21 points, win by 2)
- 🏸 Singles & Doubles support
- ⚡ Match point & Game point indicators
- 📱 Mobile responsive design
- 💾 Local storage for match history
- 🎨 Dark theme with yellow accents
